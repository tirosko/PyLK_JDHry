# Brick Blast

Small Brick Breaker style game implemented with `pygame`.

Run:

```bash
python -m BrickBlast.brick_blast
```

Install dependencies (if not already):

```bash
pip install -r requirements.txt
```

Controls:
- Left / A: move paddle left
- Right / D: move paddle right
- P: pause/unpause
- R: restart
- Esc: quit
